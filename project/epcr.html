The below are screenshots showing the portions that I have contributed to the project. The project can be toggled in either Spanish or English, but I am only showing the English version (values are stored in Spanish, however).
Create chart: top half of first page (call information)

Section headings have a background to help separate out the information associated with it. Each input field label is placed at the left of the inputs. Dropdowns are used for one selections while checkboxes are used for multiple selections. Red asterisks are used to indicate required fields.

'Mass Casualty Incident' and 'Vechicle Accident' fields can be checked to trigger for more inputs related to it.
